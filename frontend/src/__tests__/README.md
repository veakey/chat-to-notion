# Frontend Tests

## Structure

- `components/` : React component tests
- `hooks/` : Custom hooks tests

## Running Tests

```bash
# Run all tests
npm test

# Watch mode
npm test -- --watch

# With coverage
npm test -- --coverage
```

## Available Tests

### Component Tests
- `ChatPage.test.js` : Main chat component tests

### Hook Tests
- `useChatForm.test.js` : Form hook tests
